# Gym Membership App

This application is designed to manage gym memberships. It includes the following features:

- Displays a form for the user to enter their details.
- Stores user details in a database.
- Initially, the information was stored in a text file.
- The database is not yet uploaded since it was created using Derby (NetBeans).

## Note

The name of the application is tentative and may change in future updates.
